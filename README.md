# NEXT JS

This is my first NEXT JS project, I started doing it with plain React, however it was really simple and poorly done.
The project consists of a web application for Project and Task management, being done with a local MYSQL DB.
It's the first time I'm using TypeScript and Tailwind, I'm pretty much learning with practice.

## Requirements:

1. Methods to create, read, update and delete projects, tasks and users.
2. User friendly interface to apply those methods.
3. Consistency on error handling.

## Features:

### Created:

- Project API for retrieving data from the database.
- Front End layout with NavBar and Footer as a basis.
- Form for creating new projects.

-Enviroment Variables: - DB_HOST = MYSQL Host name. - DB_USER = MYSQL username. - DB_PASSWORD = MYSQL password. - DB_DATABASE = Database where the tables where created.
